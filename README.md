# Rotten Tomatoes Sentiment Classification

This repository contains a sentiment classification project comparing classical TF-IDF based models and a fine-tuned DistilBERT model on Rotten Tomatoes reviews.

Main implementation:
- rt_sentiment_classification.ipynb`

Written summary:
- rt_sentiment_report.ipynb`

## Dataset

The dataset is publicly available on Kaggle:

https://www.kaggle.com/datasets/thedevastator/movie-review-data-set-from-rotten-tomatoes

## Setup

Install required packages:

pip install -r requirements.txt

Note: GridSearchCV and DistilBERT training can be computationally intensive.
